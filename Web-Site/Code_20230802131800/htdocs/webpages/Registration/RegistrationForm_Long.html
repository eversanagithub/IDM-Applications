The Admin Portal is a secure area within the IDM Web Site where HR related administrative tasks can be easily executed.

Only those individuals whose job responsibilities align with the applications available within the portal may gain access.

The applications currently available for daily support of our HR and IDM departments are growing but presently include:.

Web Site Maintenance
