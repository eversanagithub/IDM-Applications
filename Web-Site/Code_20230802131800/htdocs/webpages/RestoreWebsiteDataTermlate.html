Website Data Recovery

 

In the event part of all of the IDM web site

data is lost or becomes corrupt, this application

can restore information from the recovery logs

and return the web site to normal operations.

 

The backup menu will allow data to be recovered

from individual tables, various dates, or both.

 

Recovery times should last no longer that a

minute and be unintrusive to normal operations.

 

Click here to initiate the data recovery operation.

 

Click here to cancel the data recovery operation.
