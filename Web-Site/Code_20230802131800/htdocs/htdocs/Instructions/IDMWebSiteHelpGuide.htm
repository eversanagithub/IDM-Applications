V.                One-Drive Delegation

a.     When an employee leaves the company, he or she also leaves behind a vacant spot that must be filled in by either an existing co-worker or a newly hired associate.

b.     To provide continuity of this vacated role, the files once owned by the former employee are shared with his or her manager so they can in-turn be passed on to the new associate.

c.      This process of sharing the former associates files with their manager is an automated task that runs each night.

d.     Times do present themselves, however, when employees other than the manager may need access to these files. This would be an exception to the process.

e.     In this event, the One-Drive Delegation application provides a means for doing just that; sharing files of a former associate with someone other than just their manager.

f.       Typically a Service-Now incident does need to be submitted for this action to happen. This ticket number can be included in the One-Drive Delegation process.

g.     To begin this process of sharing a former employees One-Drive account with another individual, click on the One-Drive Delegation button within the Admin Portal.

h.     The following screen will be presented:



i.       The top of the screen is where all user input will be received. It is a bit busy looking but we will break it down. Here are the input components:

                                                             i.      Narrow Associate Listing (For Former Associates)

1.     This area provides a means of narrowing the Former Associates drop-down listing which match the syntax in this text box.

                                                           ii.      Select Former Associates e-mail address

1.     Selection of the terminates associates e-mail address will be chosen here.

                                                         iii.      Narrow Associate Listing (For individuals requesting file access)

1.     Like the previous text box, this limits the drop-down listings of employees requesting access to the One-Drive files.

                                                         iv.      Select requester e-mail address

1.     Here is where you will select the e-mail address of the individual who is requesting access to the former associates files.

                                                           v.      Add Access

1.     Typically this is the selection you want. This selection button will grant access of the One-Drive files to the requester.

                                                         vi.      Remove Access

1.     This selection will remove access to the former associates One-Drive files from the requester once they are done with them.

2.     Normally this action will not be performed unless specifically requested by either the requester or the Legal department.

                                                       vii.      Requesting Incident Number

1.     This is where you will enter the Service-Now incident number if one was generated. Leave this area blank if no incident was generated.

                                                    viii.      Execute Request

1.     Click the Submit button to start the file sharing process.
